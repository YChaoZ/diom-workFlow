# DIOM Workflow 文档中心

## 📁 文档分类

本目录包含DIOM Workflow项目的所有技术文档，按类别组织如下：

### 📊 [reports/](./reports/) - 报告与总结

**主要文档**:

- `PROCESS_DESIGNER_SUCCESS_REPORT.md` - 流程设计器成功报告

### 📖 [guides/](./guides/) - 操作指南
快速开始、使用指南、操作手册等。

**主要文档**:
- `QUICK_START.md` - 快速开始指南
- `QUICK_TEST_GUIDE.md` - 快速测试指南
- `VITE_PROXY_EXPLANATION.md` - Vite代理详解
- `BPMN_PALETTE_GUIDE.md` - BPMN工具栏指南
- `RBAC_FRONTEND_GUIDE.md` - RBAC前端开发指南

### 🏗️ [architecture/](./architecture/) - 架构设计
系统架构、技术方案、设计文档等。

**主要文档**:
- `ARCHITECTURE_AND_CALL_CHAIN.md` - 架构与调用链
- `RBAC_DESIGN.md` - RBAC权限体系设计
- `PROCESS_DESIGNER_DESIGN.md` - 流程设计器设计
- `SERVICE_DISCOVERY_SOLUTION.md` - 服务发现方案
- `WORKFLOW_SOLUTION.md` - 工作流解决方案

### 💻 [development/](./development/) - 开发文档
开发计划、开发状态、功能进度、待办事项等。

**主要文档**:
- `TODO_LIST.md` - 待办事项列表
- `CURRENT_STATUS.md` - 当前开发状态
- `OPTIMIZATION_ROADMAP.md` - 优化路线图
- `FRONTEND_WORKFLOW_PLAN.md` - 前端工作流开发计划
